权限管理系统

基于 React + Spring Boot + Spring Security + JWT 的 权限管理系统。管理员可以编辑角色、权限和用户信息，查看用户登录情况，用户可以查看数据和修改个人信息。



## 数据库设计



登录

- 账号密码登录
- 手机号验证码登录（未查询到则默认注册新账号）

注册

- 账号密码注册
- 手机号验证码登录注册（未查询到则默认注册新账号）
